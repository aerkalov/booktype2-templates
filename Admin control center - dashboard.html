About

This is a demonstration server for Booktype. This server is reset on a regular basis, and any books created here will be deleted at that time.

How to use

If you wish to have your own installation please either download the sources http://www.sourcefabric.org/en/booktype/ or check out Booktype Pro, our free, cloud-hosted publishing service - http://sourcefabric.com/en/booktypepro/.

Need Help?
